Basic Markdown Support

Taildown fully supports standard Markdown (CommonMark) and GitHub Flavored Markdown (GFM).

Text Formatting

You can use bold text, italic text, and bold italic text.

You can also use strikethrough and inline code.

Lists

Unordered Lists

  • First item
  • Second item
  • Third item
    • Nested item
    • Another nested item

Ordered Lists

  1. First step
  2. Second step
  3. Third step
    1. Sub-step A
    2. Sub-step B

Task Lists (GFM)

  • Completed task
  • Incomplete task
  • Another task to do

Code Blocks

  console.log(`Hello, ${name}!`);
}

greet("Taildown");

# Python code block
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
