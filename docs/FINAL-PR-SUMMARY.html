Grid Layout Fix

Issue: Cards in grid rows were not equal height, violating our zero-config beauty standards.

Solution: Applied CSS Grid equal-height pattern

.component-grid {
  align-items: stretch;
}

.component-grid > * {
  height: 100%;
  display: flex;
  flex-direction: column;
}

.component-grid .component-card {
  height: 100%;
  display: flex;
  flex-direction: column;
}

Result: All cards in a row now perfectly aligned with equal heights
